# 项目简介 python56

本项目是一款基于Python的购物商城系统，采用Django框架进行开发。

## 技术栈

- 后端技术：Python、Django框架
- 前端技术：jQuery、Bootstrap、HTML、JSP
- 数据库：MySQL 5.7.33
- 开发工具：PyCharm

## 功能模块

### 前台管理

- 热卖商品
- 优惠资讯
- 投诉建议
- 后台管理
- 购物车
- 收藏商品
- 模拟支付
- 合并下单

### 后台管理

- 个人中心
- 商品分类管理
- 用户管理
- 热卖商品管理
- 投诉建议管理
- 订单管理
- 系统管理

## 系统角色

- 管理员
- 用户

## 运行环境

- Python环境
- MySQL数据库环境

## 部署步骤

1. 安装Python及依赖包
2. 安装MySQL数据库
3. 导入数据库文件
4. 配置项目环境
5. 运行项目
